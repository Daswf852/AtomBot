AtomBot
=======
### A bot for Discord written with love in Go  
  
#### Dependencies  
>discordgo by bwmarrin  
>>`go get github.com/bwmarrin/discordgo`  
  
#### Usage  
`go run AtomBot.go -t <bot token> -d <"database" file for 'seen'>`  
  
**or**  
  
`go build AtomBot.go`  
`./AtomBot -t xx -d yy`
